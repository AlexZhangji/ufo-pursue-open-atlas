> Subject: UNIDENTIFIED FLYING OBJECTS
>
> Reference is made to the instant matter which concerns the reported sighting of an unidentified flying object.
>
> A review of the photographs submitted and the various laboratory analyses conducted in connection with this matter fails to disclose any information which would lend credence to the theory that an unidentified flying object was photographed. On the contrary, the material submitted for examination was found to be consistent with that which might be obtained by use of a telescope attached to a camera in photographing an ordinary airplane in flight.
>
> It is to be noted that while it is conceded that there are various aerial phenomena for which no adequate explanation has yet been found, it would appear that the photographs submitted in this instance are not of such an unexplained phenomena.
>
> In view of the foregoing, it is recommended that this matter be closed.
